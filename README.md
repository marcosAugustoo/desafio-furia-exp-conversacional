## ⚙️ Tecnologias

- **React** (v19)
- **Vite** (v6.3)
- **TypeScript**
- **GitHub Pages / Vercel** (deploy)

## 📁 Scripts principais

- **npm run dev** – Roda a aplicação localmente.
- **npm run build** – Gera versão de produção.
- **npm run deploy** – (para deploy com GitHub Pages, se necessário).
